# ボタン

## 基本情報

- **クラス名**：`bm-c-button`
- **種類**: フォーム・コンポーネント
- **サポートしている外観**: 通常／強調／フラット／スーパーフラット
- **適用済みのStyle Element**：`bm-a-pushable`, `bm-a-focusable`

## 説明

ボタン・コンポーネントは通常の押しボタンです。HTMLの`button`要素をそのまま使用しているため、`button`要素で受け取れるパラメータは全て受け取れます。
