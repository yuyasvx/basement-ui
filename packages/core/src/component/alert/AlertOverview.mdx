# アラート

## 基本情報

- **種類**: コンポーネント
- **サポートしている外観**: -

## 説明

アラート・コンポーネントは、ユーザに対する警告・通知・確認のアイコンや文言の表示を担うコンポーネントです。

このコンポーネントはアラート表示の「中身」であるアイコン・タイトルと詳細・フッターだけで構成されています。また、警告や通知をインライン表示させたり、さりげない通知としてトースト表示させたい場合などを考慮し、表示の制御などは**担いません**。<br />
したがって、単純にアラート・コンポーネントを呼び出しただけですと、そこにアラート表示が現れるだけの状態になります。

> <Alert title="アラートのタイトル">アラートの詳細文</Alert>

モーダル表示やウインドウとしてアラート・コンポーネントを表示させたい場合は、`WindowBase`コンポーネントで括ったり、`Overlay`コンポーネントに渡すようにしてください。
