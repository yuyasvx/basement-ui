# Style Element

Style Elementは、その名の通りUIコンポーネントの「見た目」だけを提供する要素を指しています。CSSだけで定義されており、特定のCSSクラス名から呼び出すことができます。

Basement UIでは、UIコンポーネントの見た目とロジックを同時に使用することも、バラバラで使うこともサポートしていますので、ロジックは自分で実装したいが、スタイルだけBasement UIを使いたいといった場合は、Style Elementだけを呼び出すことで実現できます。

## Style Elementを使ってみる

例えば、ここになんの変哲もないdivタグがあったとします。

```
<div>Hello, world</div>
```

押しボタンの見た目を提供するStyle Elementの名前は`bm-a-pushable`と定義されいますので、上記の`div`タグのクラスに`bm-a-pushable`を付与してみましょう。

```
<div className="bm-a-pushable">Hello, world</div>
```

すると・・・

<blockquote>
  <div className="bm-a-pushable">Hello, world</div>
</blockquote>

なんとなく立体感のある押しボタンのような見た目に変化しました。

これがStyle Elementの役割です。Style Elementで定められたクラスを任意のHTMLタグに付与することで、Basement UIのボタンの見た目をすぐに適用することができます。
